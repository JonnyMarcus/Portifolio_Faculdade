# 🖥️ Arquitetura de Computadores II

## 📚 Sobre a Disciplina

Repositório destinado ao estudo dos principais conceitos de **Arquitetura de Computadores II**, abordando desde o nível ISA até arquiteturas avançadas, linguagem de montagem, segmentação de memória, aritmética computacional e processamento paralelo.

---

## 🎯 Objetivo

Aplicar os conceitos fundamentais de arquitetura de computadores na compreensão do funcionamento interno do processador, memória, instruções e execução de programas.
