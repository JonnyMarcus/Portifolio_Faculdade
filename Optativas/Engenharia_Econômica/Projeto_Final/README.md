# 📈 Sistema de Análise de Investimentos em C

## 📚 Descrição

Projeto desenvolvido em C para aplicar conceitos de Engenharia Econômica na análise de viabilidade financeira de investimentos.

---

## 🎯 Funcionalidades

* cadastro de projetos
* fluxo de caixa anual
* cálculo de Payback
* cálculo de VPL
* cálculo de TIR aproximada
* comparação entre investimentos
* salvamento em arquivo
