# 🎵 Sistema de Streaming Musical em Python

## 📚 Sobre o Projeto

Projeto desenvolvido para aplicar os principais conceitos de **Programação Orientada a Objetos**, simulando um sistema de streaming musical com organização orientada a classes, relacionamentos entre objetos e regras de negócio semelhantes a aplicações reais.

---

## 🎯 Objetivo

Aplicar os conceitos fundamentais de orientação a objetos em Python através da construção de um sistema completo de gerenciamento musical.
