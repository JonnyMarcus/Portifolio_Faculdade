# ⚙️ Engenharia de Software

## 📚 Sobre a Disciplina

Repositório destinado ao estudo dos principais conceitos de Engenharia de Software, abordando processos de desenvolvimento, levantamento de requisitos, modelagem, projeto, implementação, testes e manutenção de sistemas.

## 🎯 Objetivo

Compreender como softwares são planejados, desenvolvidos, validados e mantidos de forma organizada, utilizando processos, técnicas e boas práticas da engenharia.
