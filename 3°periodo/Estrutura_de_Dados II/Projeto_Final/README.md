# 🧠 Sistema de Gerenciamento Acadêmico em C

## 📚 Sobre o Projeto

Projeto desenvolvido para aplicar os principais conceitos de **Algoritmos e Estrutura de Dados I**, utilizando múltiplas estruturas de dados em um sistema único de gerenciamento acadêmico.

---

## 🎯 Objetivo

Integrar estruturas fundamentais estudadas na disciplina em um sistema real de manipulação de dados.

---

## 💻 Funcionalidades

### 🔹 Cadastro de alunos

* nome
* matrícula
* curso

### 🔹 Lista encadeada

* inserção
* remoção
* edição
* listagem

### 🔹 Fila de atendimento

* entrada de alunos em fila
* próximo atendimento

### 🔹 Pilha de operações

* desfazer última remoção
* desfazer última alteração

### 🔹 Árvore binária de busca

* busca rápida por matrícula
* percurso em ordem

### 🔹 Persistência em arquivo

* salvamento em arquivo `.txt`
