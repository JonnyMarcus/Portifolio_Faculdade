# 🧠 Algoritmos e Estrutura de Dados I

## 📚 Sobre a Disciplina

Repositório destinado ao estudo dos principais conceitos de **Algoritmos e Estrutura de Dados I**, com foco na implementação prática de estruturas fundamentais em linguagem C, análise de algoritmos e desenvolvimento do raciocínio lógico aplicado à organização de dados.

---

## 🎯 Objetivo

Aplicar estruturas de dados fundamentais para resolver problemas computacionais com maior eficiência, compreendendo como organizar, manipular e otimizar dados em memória.
